# 发布应用

- 帮助文档

> https://docs.serverless-devs.com/user-guide/builtin/registry/

- 登录

```
s registry login
```

- 查看列表

```shell
s registry -h
```

- 已发布列表

```shell
s registry list --private
```

- 发布新版本

```shell
s registry publish
```

- 删除版本

```shell
s registry delete --name start-docker-v3 --version-id 0.0.1
```
